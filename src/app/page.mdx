## Gabinet psychologiczno-seksuologiczny (w trakcie budowy) {{ date: '2025-07-31' }}

Strona została zaprojektowana jako nowoczesna wizytówka gabinetu psychologiczno-seksuologicznego. Aplikacja została zbudowana z wykorzystaniem Next.js i Tailwind CSS, a hosting zapewnia platforma Vercel.
Wygląd strony został przygotowany przez grafika współpracującego z klientką, a następnie wdrożony zgodnie z dostarczonym projektem.

### <SparkleIcon /> Co znajdziesz na stronie?

Witryna zawiera kilka podstron, w tym sekcję z ofertą, treści edukacyjne dotyczące zdrowego stylu życia oraz podstronę “Zapisz się”, gdzie użytkownicy mogą wypełnić formularz zapisów. Dane z formularza są przesyłane za pomocą API bezpośrednio do arkusza Google, co pozwala na prostą integrację i zarządzanie zapisami. [Zobacz stronę](https://magdalenakuznik.pl)

## Zdrowo Aktywni (w trakcie budowy) {{ date: '2024-12-30' }}

Strona Zdrowo Aktywni została stworzona dla mojej siostry przy użyciu Next.js oraz Tailwind CSS, a jej hosting odbywa się na Vercel. Projekt zawiera wiele podstron, które umożliwiają użytkownikom zapoznanie się z ofertą i treściami dotyczącymi zdrowego stylu życia. [Zobacz zdrowo-aktywni.pl](https://zdrowo-aktywni.pl)

### <SparkleIcon /> Co znajdziesz na stronie?

- O mnie - kilka słów o autorce
- Zdrowo Aktywni - główna idea i misja projektu
- Blog - artykuły na temat zdrowia, aktywności fizycznej i dobrego samopoczucia
- Sklep - produkty związane ze zdrowym stylem życia
- Współpraca - możliwości nawiązania współpracy
- Kontakt - dane kontaktowe

Strona została zaprojektowana z myślą o szybkości, wydajności i nowoczesnym designie, zapewniając użytkownikom intuicyjną nawigację i przyjemne doświadczenie.

## Pixel Picure {{ date: '2023-07-01' }}

Pixel Picture to nowoczesna platforma e-commerce stworzona z wykorzystaniem technologii Ruby, Ruby on Rails oraz Tailwind CSS. Projekt ten jest ściśle związany z moją aktualną pracą, a jego celem jest oferowanie użytkownikom płynnych, łatwych i estetycznych zakupów online, z naciskiem na intuicyjny interfejs i responsywność. [Zobacz pixelpicture.eu](https://pixelpicture.eu)

### <SparkleIcon /> Technologie użyte w aplikacji

- Ruby - język programowania, na którym bazuje cała aplikacja
- Ruby on Rails - framework, który zapewnia szybkie tworzenie aplikacji webowych i zarządzanie bazą danych
- Tailwind CSS - framework CSS, który umożliwia szybkie i elastyczne tworzenie nowoczesnych interfejsów użytkownika
- SQLite - lekka baza danych wykorzystywana do przechowywania informacji o produktach, użytkownikach i zamówieniach
- Stimulus.js - framework JavaScript do tworzenia interaktywnych komponentów na stronie
- Kamal - narzędzie do automatyzacji zadań w projekcie Rails, ułatwiające wdrażanie i zarządzanie aplikacją

Celem projektu jest stworzenie platformy, która łączy funkcjonalność z nowoczesnym designem, oferując użytkownikom prostą i przyjemną nawigację, jak i optymalizację wydajności.
